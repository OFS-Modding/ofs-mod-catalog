# Catalog format

Each mod version declares its stable id, semantic version, compatibility, dependencies, capabilities and multiplayer behavior. The downloadable `.ofmod` is identified by an immutable GitHub Release URL, exact byte count and SHA-256.

The complete technical contract lives in [`docs/catalog-format.md`](https://github.com/OFS-Modding/ofs-mod-catalog/blob/main/docs/catalog-format.md).
