# Submitting a mod

Publish the `.ofmod` as a GitHub Release asset in the mod's own repository and
compute its exact size and SHA-256. Open a **Mod submission** issue for help or
send a pull request directly. Do not commit packages or game assets to the
catalog repository.

See [`CONTRIBUTING.md`](https://github.com/OFS-Modding/ofs-mod-catalog/blob/main/CONTRIBUTING.md) for the required fields.
