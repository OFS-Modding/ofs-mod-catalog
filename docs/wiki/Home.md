# OFS Mod Catalog

This repository is the public signed discovery index for Ore Factory Squad
mods. The loader consumes its GitHub Pages endpoint. The human-facing catalog
is the [OFS Mods portal](https://ofs-modding.github.io/ofs-mod-portal/).
